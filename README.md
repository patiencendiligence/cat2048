cat2048은 고전 2048 규칙에 픽셀 아트 고양이를 얹은 퍼즐 게임입니다.

【플레이】
• 방향키 또는 WASD로 타일을 밀어 같은 숫자(같은 고양이)를 합칩니다.
• 합칠수록 더 높은 단계의 고양이로 진화합니다.
• 점수·최고 점수·도감(수집) 진행은 기기 안에만 저장됩니다.

【모드】
• 고양이 모드: 타일에 캐릭터 일러스트가 크게 보입니다.
• 숫자 모드: 숫자를 강조하고 작은 미리보기로 고양이를 볼 수 있습니다.

【개인정보】
• 사용자 데이터를 서버로 보내지 않습니다.
• 게임 설정·점수 등은 브라우저 로컬 저장소에만 보관됩니다.
• 자세한 내용은 개인정보 처리방침 링크를 참고하세요.

오프라인에 가깝게 동작하며, 별도 계정이나 로그인이 없습니다.

cat2048 is a puzzle game that combines the classic 2048 rules with pixel art cats.

【Gameplay】
• Slide tiles using the arrow keys or WASD to combine identical numbers (identical cats).

• The more you combine, the higher the cat's level evolves.

• Scores, high scores, and Pokedex (collection) progress are saved only on the device.

【Modes】

• Cat Mode: Character illustrations appear larger on the tiles.

• Number Mode: Numbers are highlighted, and you can view the cat in a small preview.

【Privacy】

• User data is not sent to a server.

• Game settings, scores, etc., are stored only in the browser's local storage.

• For more details, please refer to the Privacy Policy link.

It operates in a near-offline manner and requires no separate account or login.
